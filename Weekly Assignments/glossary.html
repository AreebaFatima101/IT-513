Web Programming Glossary

HTML
The standard markup language for creating web pages.
CSS
Used to style and layout web pages.
JavaScript
A scripting language used to add interactivity to web pages.
Element
A single component of an HTML document.
Attribute
Provides additional information about an element.
Tag
The markup used to define elements in HTML.
Browser
A software application used to view websites.
Server
A computer that hosts websites and delivers them to clients.
